What's the weather as you approach the equator?


A simple question really. Moreso a simple answer. It seems everyone knows that it's hotter at the equator. That latitude always is either dry, arid deserts or moist, tropical rain forests. But could you prove it, without a doubt, that the weather is warmer there?

This project was a chance to set that to the limit. Using a public weather API and some Python scripts, I was able to make an outputs to graph the temperature and other important weather realtaed attributes to prove the simple question.
